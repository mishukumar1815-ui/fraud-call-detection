# AI Fraud Call Detection System 🚨

## 📌 Overview

This project detects fraudulent calls in real-time using AI-based keyword analysis and simulated NLP.

## 🚀 Features

* Real-time call analysis
* Fraud keyword detection (OTP, bank, payment, etc.)
* Risk classification (Safe / Suspicious / Fraud)
* Interactive UI with logs

## 📁 Project Structure

* frontend/ → UI (React + Vite)
* backend/ → Python API

## ▶️ How to Run

### Frontend:

cd frontend
npm install
npm run dev

### Backend:

cd backend
pip install -r requirements.txt
python main.py
